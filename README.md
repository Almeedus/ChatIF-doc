# ChatIF-doc

Este repositório contém o Trabalho de Conclusão de Curso (TCC) intitulado “ChatIF”. O documento foi desenvolvido como requisito para a obtenção do título de Especialista em Computação Aplicada à Educação pelo Instituto Federal de Educação, Ciência e Tecnologia de São Paulo (IFSP), Câmpus Itapetininga.

## Tecnologias Usadas
- Frontent: VueJS
- Backend: Python
- Banco de dados: Redis
- Criação da monografia: LaTex

## Compilação
Para visualizar o documento final, basta abrir o arquivo utfpr-tcc.pdf.

Caso deseje compilar o documento-fonte em LaTeX, os arquivos necessários estarão na pasta correspondente. Será preciso ter uma distribuição LaTeX instalada (como MiKTeX ou TeX Live).
